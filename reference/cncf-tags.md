# CNCF Technical Advisory Groups (TAGs)

<!-- Verified: 2026-04-26 — see scripts/verify_repo.py and re-confirm against cncf.io/people/technical-advisory-groups/ -->

In May 2025, the CNCF TOC restructured from 8 TAGs to 5. TAG App Delivery, TAG Runtime, and TAG Environmental Sustainability no longer exist. Old Slack channels were archived in June 2025.

**Important:** CNCF TAGs are advisory groups — they don't own code. Don't confuse them with Kubernetes SIGs, which do own code. See [`governance-quick-ref.md`](governance-quick-ref.md) for the distinction.

## What Was Retired (May 2025)

| Former TAG | Status |
|-----------|--------|
| TAG App Delivery | Retired — scope absorbed into other TAGs |
| TAG Runtime | Retired — scope absorbed into TAG Workloads Foundation |
| TAG Environmental Sustainability | Retired — scope absorbed into TAG Operational Resilience |
| TAG Network | Retired — scope absorbed into TAG Infrastructure |
| TAG Storage | Retired — scope absorbed into TAG Infrastructure |
| TAG Contributor Strategy | Retired |
| TAG Observability | Retired — scope absorbed into TAG Operational Resilience |
| TAG Security | Reconstituted as TAG Security and Compliance |

## TAG Slack Is on CNCF Slack, Not Kubernetes Slack

TAGs use [CNCF Slack](https://cloud-native.slack.com) (separate from Kubernetes Slack at slack.k8s.io). You need separate accounts for each workspace.
